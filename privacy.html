Politique de confidentialité

Dernière mise à jour : 9 avril 2026 · Conforme RGPD

🔒
Engagement principal : eDataFoot ne collecte, ne stocke et ne revend aucune donnée personnelle vous concernant via ce site web. Ce site est une simple page de redirection statique.

1. Qui sommes-nous ?

eDataFoot est un projet indépendant d'analyse et de prédiction sportive basé sur des algorithmes d'intelligence artificielle. Ce site web statique est hébergé sur GitHub Pages (service de Microsoft/GitHub).

Contact : canal Telegram public @efootdata

2. Données collectées par ce site

Ce site web ne collecte aucune donnée personnelle de manière active. Il n'y a :

  • Aucun formulaire d'inscription
  • Aucun cookie de traçage ou publicitaire
  • Aucun pixel de suivi (Meta Pixel, Google Analytics, etc.)
  • Aucune base de données utilisateurs
  • Aucun compte à créer

3. Données collectées par les services tiers

4. Abonnement VIP (Stripe + Telegram)

Si vous souscrivez à l'offre VIP, les données de paiement (nom, email, coordonnées bancaires) sont traitées exclusivement par Stripe, opérateur certifié PCI-DSS. eDataFoot ne stocke jamais vos données bancaires.

Seul votre identifiant Telegram est conservé dans notre base de données pour gérer votre accès au canal privé. Cet identifiant n'est pas partagé avec des tiers.

5. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : savoir quelles données nous détenons sur vous
  • Droit de rectification : corriger des données inexactes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit à la portabilité : récupérer vos données dans un format lisible

Pour exercer ces droits, contactez-nous sur Telegram : @efootdata. Nous répondrons dans un délai de 30 jours.

6. Cookies

Ce site n'utilise aucun cookie de son propre chef. GitHub Pages peut placer des cookies techniques nécessaires au fonctionnement de l'hébergement. Aucun cookie publicitaire n'est déposé.

7. Sécurité

Ce site est servi via HTTPS. Bien que nous ne collections pas de données personnelles, nous appliquons les bonnes pratiques de sécurité web recommandées.

8. Modifications

Cette politique peut être mise à jour à tout moment. La date en haut de page indique la version en vigueur. L'utilisation continue du service vaut acceptation.
