# 🚀 Mettre ta page eDataFoot en ligne sur GitHub Pages (GRATUIT)

URL finale : `https://TON-PSEUDO.github.io/edatafoot/`

---

## Étape 1 — Créer un compte GitHub (si pas déjà fait)

1. Va sur https://github.com/signup
2. Crée un compte gratuit

---

## Étape 2 — Créer un nouveau dépôt

1. Va sur https://github.com/new
2. Remplis :
   - **Repository name** : `edatafoot`
   - **Visibility** : ✅ Public (obligatoire pour GitHub Pages gratuit)
   - **Add a README file** : laisse décoché
3. Clique **"Create repository"**

---

## Étape 3 — Uploader les fichiers

### Option A — Via l'interface web (le plus simple)

1. Dans ton dépôt, clique **"Add file" → "Upload files"**
2. Glisse-dépose ces 3 fichiers :
   - `index.html`
   - `terms.html`
   - `privacy.html`
   - `profile.jpg` ← ta photo de profil (optionnel)
3. Clique **"Commit changes"**

### Option B — Via Git (terminal)

```bash
cd C:\Users\bouzi\foot_predictor\github_pages
git init
git add .
git commit -m "Initial commit - eDataFoot landing page"
git remote add origin https://github.com/TON-PSEUDO/edatafoot.git
git branch -M main
git push -u origin main
```

---

## Étape 4 — Activer GitHub Pages

1. Dans ton dépôt GitHub, clique sur **"Settings"** (onglet en haut)
2. Dans le menu gauche, clique **"Pages"**
3. Sous **"Source"** :
   - Branch : **main**
   - Folder : **/ (root)**
4. Clique **"Save"**

⏳ Attends 1-2 minutes...

5. Un bandeau vert apparaît :
   **"Your site is live at https://ton-pseudo.github.io/edatafoot/"**

---

## Étape 5 — Personnaliser la photo de profil

1. Renomme ta photo **`profile.jpg`**
2. Upload-la dans le dépôt (même méthode que les HTML)
3. La page se met à jour automatiquement

---

## Étape 6 — Mettre ton lien TikTok

Dans la bio TikTok, mets :
```
https://ton-pseudo.github.io/edatafoot/
```

TikTok autorise ce type de lien externe (domaine GitHub = lien de confiance).

- Clique l'icône crayon ✏️
- Modifie et "Commit changes"
- La mise à jour est en ligne en moins d'1 minute !
